# Brain Firmwares

This is a collection of firmwares for the [Brain module](https://github.com/shmoergh/moduleur/tree/main/modules/07-brain), released as part of the [Shmoergh Moduleur](https://www.shmoergh.com/moduleur/).

## DIY tools
- [Brain Diagnostics Tool](https://github.com/shmoergh/brain-diagnostics) — Tools to test the hardware of the Brain module. Useful if you're building the module yourself and you want to make sure everything works as expected.
- [Brain Output Tuner](https://github.com/shmoergh/brain-output-tuner) — Another tool for DIY builders that helps fine tuning the Brain outputs so they track precisely on the 1V/oct scale.

## MIDI
- [Basic MIDI2CV converter](https://github.com/shmoergh/brain-basic-midi2cv) — Single channel MIDI to CV converter.

## CV
- [CV utilities](https://github.com/shmoergh/brain-cv-utils) — 6 CV utilities packed in a single program.
